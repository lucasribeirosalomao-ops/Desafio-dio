# 🏆 Calculadora de Partidas Rankeadas

## 📌 Descrição

Projeto desenvolvido como desafio da DIO com o objetivo de aplicar conceitos fundamentais de lógica de programação em JavaScript.

A aplicação calcula o saldo de vitórias de um jogador e determina seu nível com base em regras pré-definidas.

## 🚀 Tecnologias utilizadas

* JavaScript
* Node.js

## ⚙️ Funcionalidades

* Cálculo do saldo de vitórias (vitórias - derrotas)
* Classificação do jogador em níveis:

  * Ferro
  * Bronze
  * Prata
  * Ouro
  * Diamante
  * Lendário
  * Imortal

## 🧠 Regras de classificação

| Saldo de Vitórias | Nível    |
| ----------------- | -------- |
| até 10            | Ferro    |
| 11 a 20           | Bronze   |
| 21 a 50           | Prata    |
| 51 a 80           | Ouro     |
| 81 a 90           | Diamante |
| 91 a 100          | Lendário |
| 101+              | Imortal  |

## ▶️ Como executar

```bash
node index.js
```

## 💡 Exemplo de saída

```bash
O Herói tem saldo de 55 e está no nível Ouro
```
